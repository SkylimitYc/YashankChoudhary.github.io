# 🚀 DevOps Portfolio Deployment

This project demonstrates deployment of a portfolio website using Docker and Nginx.

## 🔧 Tech Stack
- HTML, CSS, JavaScript
- Docker
- Nginx

## ⚙️ Steps Performed
- Created Dockerfile
- Built Docker image
- Ran container using Docker
- Exposed port 8080
- Hosted website locally

## ▶️ Run Locally

docker build -t my-portfolio .
docker run -d -p 8080:80 --name portfolio my-portfolio

## 🌐 Output
http://localhost:8080

## 🎯 Goal
Learning DevOps, deployment, and containerization.
